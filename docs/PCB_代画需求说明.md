# PCB 代画需求说明书（最简版）— MKR WiFi 1010 传感器盾板

> 发给嘉立创代画/工程师时，把这份 + `docs/pcb_schematic_v1.pdf` 一起发过去。

## 1. 项目一句话说明
为 **Arduino MKR WiFi 1010** 做一块**扩展盾板（shield）**：
把 **VL53L0X（测距）和 BME280（温湿度气压）两个现成模块直接焊在板上**，走 I²C；
**RC522（RFID 读卡）外接**，板上只留接口。无任何其它预留口。单件手工焊接。

## 2. 板子类型与机械要求
- 类型：**MKR WiFi 1010 扩展盾板**，通过两排 2.54mm 母排座插在 MKR 上。
- 主控封装：用**官方 Arduino MKR WiFi 1010 footprint**（嘉立创/EasyEDA 库里有，2×14P）。
- 板框：约 **61.5 × 25 mm**，与 MKR 同宽，能放下两个传感器模块和 RC522 接口即可。
- 双面板，板厚 1.6mm。

## 3. 引脚分配表（严格按此，和固件一致）
| 功能 | MKR 引脚 | 去向 |
|---|---|---|
| 电源 3.3V | VCC(3V3) | 所有模块 VCC |
| 地 GND | GND | 所有模块 GND |
| I²C SDA | D11 | VL53L0X + BME280 |
| I²C SCL | D12 | VL53L0X + BME280 |
| SPI MOSI | D8 | RC522 MOSI |
| SPI SCK | D9 | RC522 SCK |
| SPI MISO | D10 | RC522 MISO |
| RC522 片选 SS | D5 | RC522 SDA/SS |
| RC522 复位 RST | D4 | RC522 RST |

## 4. 板载 / 外接清单
| 位号 | 器件 | 安装方式 | 引脚 |
|---|---|---|---|
| U1 | **VL53L0X 模块** | **焊在板上** | VCC · GND · SDA(D11) · SCL(D12) |
| U2 | **BME280 模块** | **焊在板上** | VCC · GND · SDA(D11) · SCL(D12) |
| J1 | **RC522 接口** | 板上 7 针排针，**模块外接** | 3V3 · GND · SS(D5) · SCK(D9) · MOSI(D8) · MISO(D10) · RST(D4)（IRQ 空）|

## 5. 元件清单（BOM）
- 排母 ×2：MKR WiFi 1010 用，2×14P，2.54mm。
- VL53L0X 模块 ×1（焊在板上）。
- BME280 模块 ×1（焊在板上）。
- 排针 ×1：RC522 接口 7P，2.54mm。
- **无上拉电阻**（模块自带上拉）。**无贴片、不需要 PCBA 服务**，全部手工焊接。

## 6. 特殊要求（务必告知工程师）
1. VL53L0X、BME280 两个**现成模块直接焊在 PCB 上**（按模块排针间距开孔/焊盘），不是用连接器外接。
2. **RC522 外接**：板上只放 J1 这个 7 针接口，**放在板子边缘**，**接口附近不要铺铜**（RFID 13.56MHz 天线对金属敏感），模块用短线引到板外。
3. **不放上拉电阻**，**不留任何其它预留口**（不要 OLED/WS2812B/PIR/TEMT6000）。
4. 全部 **3.3V 供电**，不接 5V。
5. **丝印**标清每个接口名称和针脚，方便手工接线。

## 7. 制板工艺参数（下单时填）
- 层数：双面（2 层）　板厚：1.6mm
- 阻焊颜色：自选　表面处理：喷锡 HASL（便宜）即可
- 数量：5 片（起订量）　运输：DHL Express（约 5–7 天到英国）
